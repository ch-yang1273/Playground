호버 효과

호버해보세요!

생성되는 CSS 예시

.blur-sm {
  --tw-blur: blur(4px);
  filter: var(--tw-blur,) var(--tw-brightness,) var(--tw-grayscale,) var(--tw-contrast,);
}

.grayscale {
  --tw-grayscale: grayscale(100%);
  filter: var(--tw-blur,) var(--tw-brightness,) var(--tw-grayscale,) var(--tw-contrast,);
}

.brightness-75 {
  --tw-brightness: brightness(0.75);
  filter: var(--tw-blur,) var(--tw-brightness,) var(--tw-grayscale,) var(--tw-contrast,);
}

각 클래스는 동일한 CSS를 생성하지만, CSS 변수를 통해 조합된 효과가 적용됩니다.
